設定
YouTube Data API v3 キーを設定すると、トレンド動画を見たり、動画を検索・再生したりできます。キーは無料で取得できます。
📱 iPhoneでアプリとして使う方法
1
このページをBraveで開く
2
上のGoogleボタンでサインインするか、APIキーを入力
3
画面下の 共有ボタン（□↑） をタップ
4
「ホーム画面に追加」をタップ
5
ホーム画面のアイコンから起動 → アドレスバーが消える！
🔧 API キーの取得方法（手動の場合）
1
console.cloud.google.com を開く
2
新規プロジェクト作成
3
「APIとサービス」→「ライブラリ」→「YouTube Data API v3」有効化
4
「認証情報」→「APIキー」をコピー
