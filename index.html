設定
YouTube Data API キーを設定すると、トレンド動画の閲覧・検索ができます。取得は無料です。
🔑 APIキーの無料取得方法
1
ブラウザで console.cloud.google.com を開く（Googleアカウントが必要）
2
「新しいプロジェクト」を作成する
3
「APIとサービス」→「ライブラリ」→「YouTube Data API v3」を有効化
4
「認証情報」→「認証情報を作成」→「APIキー」をクリック
5
生成されたキーをコピーして上に貼り付け「保存」
💡
1日10,000リクエストまで無料。検索1回 = 100消費。
📱 iPhoneでアプリとして使う方法
1
このページをBraveで開く
2
画面下の 共有ボタン（□↑） をタップ
3
「ホーム画面に追加」をタップ
4
ホーム画面のアイコンから起動するとアドレスバーが消えてアプリ風になります
